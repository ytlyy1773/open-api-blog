## <OptionBadge type="get" />user/:id

> 返回id对应的用户数据，不存在返回默认用户数据

### 接口地址

```url
https://www.openapijs.com/api/user/:id
```

### 入参

**id**

取值范围1-8

### 响应示例

```json
{
    "data": {
        "name": "比尔·盖茨",
        "year": 1955,
        "sex": "男",
        "nationality": "美国",
        "school": "哈佛大学",
        "achievement": [
            "创办微软公司",
            "成立比尔和梅琳达·盖茨基金会",
            "主要作品有《走进比尔：解码比尔·盖茨》"
        ]
    },
    "msg": "success",
    "code": 200
}
```

### Attributes

<OptionTable
  options={[
    ['id', 'string', '返回对应用户id的用户数据']
  ]}
/>

## <OptionBadge type="post" />user

### 接口地址

```url
https://www.openapijs.com/api/user
```

> 返回所有的用户信息列表

### 入参

```json
{
  "id" : "2"
}
```

### 响应示例

```json
{
    "data": {
        "name": "雷军",
        "year": 1969,
        "sex": "男",
        "nationality": "中国",
        "school": "武汉大学",
        "achievement": [
            "带领金山软件上市",
            "带领小米科技上市",
            "主要作品有《城市梦想》"
        ]
    },
    "msg": "success",
    "code": 200
}
```

### Attributes

  options={[
    ['id', 'string?', '返回对应用户id的用户数据']
