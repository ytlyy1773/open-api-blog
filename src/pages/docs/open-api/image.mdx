# 随机图片

随机从图库获取一张图片

## <OptionBadge type="get" />random/image

### 接口地址

```url
https://www.openapijs.com/api/random/image
```

**curl**

```bash
curl --location 'https://www.openapijs.com/api/random/image'
```

> 随机返回图库中的一张图片

### 响应示例

```text
https://www.jwblog.cn/images/openapi/animal/animal003.jpg
```

## <OptionBadge type="post" />random/image

### 接口地址

```url
https://www.openapijs.com/api/random/image
```

**curl**

```bash
curl --location --request POST 'https://www.openapijs.com/api/random/image'
```

> 返回特定数量随机图片数组

### 入参

```json
{
  "length": 10
}
```

### 响应示例

```json
{
  "data": [
    "https://www.jwblog.cn/images/openapi/animal/animal010.jpg",
```
