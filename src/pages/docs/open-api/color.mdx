# 随机颜色

随机生成一个颜色

## <OptionBadge type="get" />color

### 接口地址

```url
https://www.openapijs.com/api/color
```

**curl**

```bash
curl --location 'https://www.openapijs.com/api/color'
```

> 返回所有的随机数据

### 响应示例

```json
{
    "data": "#8D98AF",
    "msg": "success",
    "code": 200
}
```

### Attributes

<OptionTable
  options={[
    ['letters', 'string?', '颜色十六进制字符字符串', '0123456789ABCDEF']
  ]}
/>

## <OptionBadge type="post" />color

### 接口地址

```url
https://www.openapijs.com/api/color
```

**curl**

```bash
curl --location 'https://www.openapijs.com/api/color' \
--header 'Content-Type: application/json' \
--data '{
    "letters": "89A",
    "count": 10
}'
```

> 返回所有的随机数据

### 响应示例

```json
{
    "data": [
        "#89A9A8",
        "#AA8998",
        "#988AAA",
        "#9A8988",
        "#A988A8",
        "#A89A9A",
        "#98A8A8",
        "#898888",
        "#998999",
        "#9998A8"
    ],
    "msg": "success",
    "code": 200
}
```

### Attributes

<OptionTable
  options={[
    ['letters', 'string?', '颜色十六进制字符字符串', '0123456789ABCDEF'],
    ['count', 'number?', '随机颜色的数量', '10']
  ]}
/>
