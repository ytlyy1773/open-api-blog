# 随机文字

包括随机数字、随机明星、随机诗文、随机dom模版

## <OptionBadge type="get" />random

### 接口地址

```url
http://localhost:4000/random
```

> 返回所有的随机数据

## <OptionBadge type="post" />random

### 接口地址

```url
http://localhost:4000/random
```

> 返回对应的随机数据

### 入参

```json
{
    "number": true,
    "poems": true
}
```

### 响应示例

```json
{
    "data": {
        "number": 7048,
        "poems": {
            "title": "清明",
            "autho": "杜牧",
            "textArray": [
                "清明时节雨纷纷，",
                "路上行人欲断魂。",
                "借问酒家何处有？",
                "牧童遥指杏花村。"
            ],
            "domText": "<p>清明时节雨纷纷，</p><p>路上行人欲断魂。</p><p>借问酒家何处有？</p><p>牧童遥指杏花村。</p>"
        }
    },
    "msg": "success",
    "code": 200
}
```
