## 开源 Api 接口

提供简单、开源、免费的 api 接口

供广大开发者学习使用

## 技术架构

- 前端使用 [nextjs](https://nextjs.org/) 开发

- 后端使用 [nestjs](https://nestjs.com/) 开发

## 系统要求

- **Node.js 18.18** 或更高版本。

## 运行

> 国内建议使用 `npm` 淘宝镜像地址下载

```bash
pnpm i

pnpm dev
```
